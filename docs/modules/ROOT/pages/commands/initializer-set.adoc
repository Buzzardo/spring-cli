= initializer set
Set the Initializr server environment

== NAME
initializer set

== SYNOPSIS
====
[source]
----
initializer set --server-id String --url String --help 
----
====

== OPTIONS
`--server-id` String Server to use *Optional*

`--url` String Server base url *Optional*

`--help or -h`  help for initializer set *Optional*
